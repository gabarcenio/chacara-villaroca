# Chácara VillaRoça Booking

Aplicativo web em Next.js para solicitação de orçamento e reserva da Chácara VillaRoça.

## Rodando localmente

```bash
npm install
npm run dev
```

## Verificações

```bash
npm run lint
npm run typecheck
npm run build
```

## Fatia atual

- Landing mobile-first preservando o visual do mockup da Figma.
- Calendário com regra de sexta, sábado e domingo.
- Formulário público de orçamento com validação de capacidade.
- Solicitações salvas localmente como prévia até a conexão com Supabase.
- Rotas públicas principais, painel de conta e placeholders administrativos.
